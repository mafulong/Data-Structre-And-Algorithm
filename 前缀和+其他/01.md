### 1. Number of Ways

[原题链接](https://codeforces.com/problemset/problem/466/C)

![image-20200913234035287](images/01.png)

#### 解题思路

首先求出所有数字的和，如果不能被3整除则直接返回0

然后问题就转化为求`i, j` 对使其满足：

- `i < j`
- `s[i] = sum / 3`
- `s[j] = sum / 3 * 2`

只需要一趟扫描，记录满足和为三分之一的位置的个数，如果当前和为三分之二，则加上其之前和为三分之一的个数即可。

#### C++代码

```c++
#include <iostream>
using namespace std;
typedef long long LL;
const int N = 5e5 + 10;

LL s[N];

int main() {

    int n;
    cin >> n;
    LL ans = 0;
    for (int i = 1; i <= n; i ++) {
        cin >> s[i];
        s[i] += s[i-1];
    }
    if (s[n] % 3) cout << 0;
    else {
        LL one = s[n] / 3, two = one * 2;
        int cnt = 0;
        for (int i = 1; i < n; i++) {
            if (s[i] == two) ans += cnt;
            if (s[i] == one) cnt++;
        }
        cout << ans;
    }
    return 0;
}
```
